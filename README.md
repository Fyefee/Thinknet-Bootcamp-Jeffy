#### How to run on development
<br>
1. create .env file and setup environment
<br>

```
PORT=3000
MONGOURL=mongodb+srv://<YourDatabase>:<Password>@cluster0.xfiz9.mongodb.net/<CollectionName>?retryWrites=true&w=majority
```
<br>
2. run on development
<br>

```
yarn start
```
